Question 1
Which command line utility would you use to determine the following:

The spatial extent of a GML file?

You would use ogr (ogrinfo) in order to determine the spatial extent of a GML file.

The spatial reference system of a raster file in the format of a GeoTIFF?

You would use gdal (gdalinfo) in order to determine the spatial reference system of a raster file/GeoTIFF.

Question 2
What combination of OGC Service and Request (e.g. SERVICE=WMS&REQUEST=GetFeatureInfo) would you use to perform the following:

1) Determine the spatial extent of an available layer from a Web Map Service?

SERVICE=WMS&REQUEST=GetCapabilities.

2) Obtain a list of coverages from a Web Coverage service?

SERVICE=WCS&REQUEST=GetCapabilities.

3) Determine what file formats are supported by a Web Feature Service for the delivery of available data types (i.e. layers)?

SERVICE=WFS&REQUEST=GetCapabilities.

4) Retrieve a map image from a Web Map Service?

SERVICE=WMS&REQUEST=GetMap.

5) Retrieve data from an available coverage from a Web Coverage Service?

SERVICE=WCS&REQUEST=GetCoverage.

Question 5
Compose a GetMap request for the WMS referenced in Question 4 that includes the following characteristics:

  • JPEG image format
  • 1200 pixels wide (calculate the height based upon the aspect ratio of the bounding box)
  • Bounding Box (EPSG:4326)
  • Min X = -106.993 East Longitude
  • Min Y = 34.989 North Latitude
  • Max X = -106.243 East Longitude
  • Max Y = 35.489 North Latitude
  • Layer to be mapped = “SurveyAreaPoly”
  • Include the complete WMS GetMap request and the resulting map image that is returned.

To define the height:

BBOX width = (maxx) - (minx) = (-106.243)-(-106.993) = 0.75
BBOX height = (maxy) - (miny) = (35.489)-(34.989) = 0.5
BBOX aspect ratio = (BBOX width) / (BBOX height) = 0.75/0.5 = 1.5
width = 1200 pixels
height = 1200/1.5 = 800 pixels

WMS GetMap Request:

http://SDMDataAccess.nrcs.usda.gov/Spatial/SDM.wms?VERSION=1.1.1&SERVICE=wms&REQUEST=GetMap&SRS=EPSG:4326&BBOX=-106.993,34.989,-106.243,35.489&LAYERS=SurveyAreaPoly&WIDTH=1200&HEIGHT=800&FORMAT=image/jpeg&STYLE=
