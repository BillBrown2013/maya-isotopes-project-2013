Question 1: What request type is common across all three (WMS, WFS, WCS) OGC web services that we have learned about?

Answer:
GetCapabilities is a request used by all three OGC web services.

Question 2: What are the required parameters, and what do they represent?
What is returned in response to a WMS GetCapabilities request?

Answer:
1) REQUEST=GetCapabilities. This command is used when you want a service to tell you what capabilities it provides.
2) SERVICE=WMS. This represents the service type for the request being submitted.

Question 3: What are the required parameters and what do they represent?
What is returned in response to a WMS GetMap request?
What is the significance of transparency in WMS requests?

Answer:
1) REQUEST=GetMap: This tells the service that you want a map.
2) BBOX: The BBOX parameter defines the area of interest for the map by setting the bounding box for the map you want.
3) FORMAT: The format of the image (i.e., jpeg, png, etc.) is needed for the map request.
4) VERSION: This represents the WMS version being used for the map request.
5) LAYERS: Represents the layers of a map that will be rendered in the map image you want.
6) SRS: The SRS is the spatial reference system for the map (i.e., WGS, NAD83, etc.).
7) WIDTH and HEIGHT: Represents how high and wide (in pixels) you want you map.
8) STYLES: Represents the style used when returning the map.

When a WMS GetMap request is submitted, if you are lucky you get an image of the map you want, but if you have submitted incorrectly, you will get an error message.

If your request includes the parameter TRANSPARENT=TRUE the background pixels will be transparent (if the layers and the file format support transparency).

The tranparency allows the layers stacked below the upper layer to be visible in a client application.

Question 6: What are the EPSG codes of the following Spatial Reference Systems
WGS 84 (Geodetic CRS [geographic 2d])
NAD83 / UTM zone 13N
NAD27 / UTM zone 13N

Answer:
WGS 84 (Geodetic CRS [geographic 2d]) EPSG:4326.
NAD83/UTM Zone 13N EPSG:26913
NAD27/UTM Zone 13N EPSG:26713
